## データ構造（タスクオブジェクト）

```javascript
{
  id: "timestamp_randomstr",      // ユニークID
  name: "タスク名",                // 必須
  memo: "メモ（改行対応）",         // 任意
  deadline: "2026-03-15",         // 任意（YYYY-MM-DD）
  parentId: "親タスクのid",        // 任意
  scheduledDate: "2026-03-10",    // 任意（先の予定タブ用）
  status: "todo",                 // "todo" / "doing" / "done"
  todayDate: "2026-03-09",        // 今日のボードに追加された日
  completedDate: "2026-03-09",    // 完了した日（日記表示に使用）
  isRoutine: false,               // ルーティンフラグ
  routineDoneToday: "2026-03-09", // ルーティン当日チェック済みの日付
  directTask: false               // 「今日やる」直接入力フラグ ★2026-03-09追加
}
```

### ⚠️ `directTask` フラグについて
- `true` = 「今日やる」欄に直接入力したタスク
- 翌日も未完了なら「今日やる」に自動持ち越し
- 倉庫・倉庫から選ぶには表示されない
- `false`（デフォルト）= 倉庫から☀️で追加 → 翌日は倉庫に戻る

## タスクの動き一覧

| タスク種類 | 今日やる中 | 翌日未完了 | 完了後 |
|---|---|---|---|
| 直接入力（directTask:true） | 倉庫に出ない | 今日やるに残る | 日記へ |
| 倉庫から☀️ | 倉庫から消える | 倉庫に戻る | 日記へ |
| ルーティン | 倉庫に出ない・独立 | チェックのみリセット | 日記には出ない |
| 先の予定 | 当日になると自動で今日やるへ | 倉庫に戻る | 日記へ |

## 画面構成（4タブ）

タブ順（左から）※デフォルトは「今日のボード」:
1. ☀️ 今日のボード
2. 🗓️ 先の予定
3. 📦 倉庫
4. 📖 日記

---

### ☀️ 今日のボード

**レイアウト:**
- PC: 左に「倉庫から選ぶ」パネル（sticky）/ 右に3列ボード
- iPhone: 上に3列ボード / 下に「倉庫から選ぶ」パネル

**倉庫から選ぶパネル（黄色系）:**
- 倉庫のタスク＋子タスクを表示（directTask・完了・今日追加済みは除外）
- ☀️ボタンで「今日やる」列に追加
- メモありタスク: PCはホバーで吹き出し、iOSはロングタップでポップアップ

**3列ボード:**

| 列 | 色 | 機能 |
|----|-----|------|
| 🔁 ルーティン | サーモン | 毎日表示。チェックのみ翌日リセット。✎編集ボタンあり |
| ☀️ 今日やる | 黄色 | 直接入力 or 倉庫から追加。✎編集ボタンあり。PC/iOSドラッグ並び替え可 |
| ✅ 完了 | ターコイズ | チェックで自動移動。✕で今日やるに戻す |

---

### 🗓️ 先の予定

- **明日〜7日後**を表示（今日は除外）
- 左パネル: 倉庫から選ぶ（sticky）。ドラッグで日付にセット可（PC・iOS対応）
- 右パネル: 各日付ごとにタスク表示。日付内で並び替え可・日付間移動可
- 当日になると自動で「今日やる」に移動
- ✕ボタンは削除ではなく**倉庫に戻す**
- メモありタスク: PCはホバー、iOSはロングタップで表示

---

### 📦 倉庫

**1カラム・折りたたみ式トグル表示（2026-03-09変更）**
- 子タスクがある親は **▶** → タップで展開（**▼**）、再タップで折りたたみ
- 子タスクなしは単独表示
- 各タスクに: ☀️（今日やるへ）/ ＋（子タスク追加）/ ✎（編集）/ ✕（削除）
- ✕ボタン: scheduledDate・todayDateがない場合のみ完全削除（確認あり）
- PC: ドラッグで並び替え可

**倉庫に表示されないタスク:**
- ルーティン（isRoutine:true）
- 完了済み（status:'done'）
- 直接入力タスク（directTask:true）
- 今日のボードに追加済み（todayDate===今日）
- 先の予定に設定済み（scheduledDate あり）

---

### 📖 日記

- 左パネル（黄色系）: カレンダー（完了タスクがある日にドット表示）
- 右パネル（サーモン系）: 完了タスク一覧・メモ（自動保存）・画像添付（localStorageのみ）

## メモ表示機能（2026-03-09追加）

メモがあるタスクに対して:
- **PC（hover対応環境）**: `.task-content` にカーソルを合わせると黒い吹き出しで表示
- **iOS**: タスクを500ms長押しでポップアップ表示、他をタップで閉じる
- 対応箇所: 今日のボード（今日やる・完了列・倉庫から選ぶ）、先の予定（タスク・倉庫から選ぶ）

```javascript
attachMemoTip(el, memo) // elは.task-content要素
```

---

## モーダル仕様（2026-03-09変更）

- **枠外クリックで閉じない**（アイデアが消えるのを防ぐため）
- キャンセルまたは保存ボタンでのみ閉じる
- メモ欄は `<textarea>` で入力量に合わせて自動リサイズ・改行対応
- タスク名欄でEnterキーを押すと保存

## 運用・注意事項

- **Firebase Sparkプラン（無料）**: 1日5万読み取り・2万書き込みまで。個人利用では超えない
- **Firestoreセキュリティルール**: 現在テストモード（将来要対応）
- **画像は基本添付しない運用推奨**（localStorageの容量節約）
- **データのバックアップ**: ヘッダーの「💾書き出し」ボタンでJSON保存
- **複数デバイス同期**: 同じURLを開くだけ。ページを開くたびに自動同期
- **オフライン**: 現状非対応（Service Worker未実装）
